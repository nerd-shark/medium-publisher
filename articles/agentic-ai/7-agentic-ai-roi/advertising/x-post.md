## Thread

1/ The full cost stack for a production AI agent:

• LLM API: $500-2K/mo (5-15%)
• Infrastructure: $1.5-5K/mo (15-25%)
• Engineering maintenance: $3-6K/mo (25-40%)
• Human review: $2-7K/mo (20-35%)

Total: $8-20K/mo. Not the $800 from the pilot proposal.

2/ Where agents deliver real ROI:
- High-volume, repetitive tasks (50K+ interactions/mo)
- Tasks where speed prevents losses (fraud, security triage)
- Tasks with expensive error costs the agent reduces

Where they lose money:
- Low volume (<5K/mo)
- Tasks requiring 40%+ human review
- "AI for AI's sake" projects

3/ The single biggest variable in agent ROI: volume.

At 10K docs/mo, an agent costs $1.50/doc vs $6.80 for a human. Strong ROI.

At 1K docs/mo, the same agent costs $15/doc. You're losing money.

Fixed costs don't scale down. Know your volume before you build.
