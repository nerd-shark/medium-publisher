Customer calls about billing. Agent resolves it. Two weeks later, same customer, related issue.

Agent has zero memory. Asks the same questions from scratch.

Customer: "I already explained this."

Your agent is a goldfish. Context window closes, everything evaporates.

Fix: vector databases for long-term memory. Store by meaning, not keywords. "Billing problem with card 4521" finds the previous interaction even with different wording.

And compress old memories into summaries. One useful summary beats 200 individual records.
