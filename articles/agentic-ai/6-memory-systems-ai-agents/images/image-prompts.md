# Featured Image Prompts — Memory Systems for AI Agents

## Prompt 1: The Three Memory Layers
A dramatic illustration of a futuristic AI brain shown in cross-section with three distinct glowing layers. The outermost layer (bright, electric blue, thin) represents working memory — fast, temporary, currently active data streams flowing rapidly. The middle layer (warm amber, medium thickness) represents short-term memory — session data, intermediate results, organized in structured patterns. The innermost core (deep purple/violet, large, dense) represents long-term memory — vast, deep, with crystalline structures representing consolidated knowledge. Data flows between layers — bright streams moving inward from working to short-term, and slower, thicker streams consolidating from short-term to long-term. Small memory fragments (glowing particles) are being retrieved from the deep core and flowing outward to the working layer. Style: anatomical cross-section meets sci-fi, glowing layers on dark background. 16:9 landscape. No text, no labels.

## Prompt 2: The Goldfish vs The Expert
A split-scene illustration. Left side: a simple, transparent AI robot (goldfish-like, with a small glowing brain) sits at a desk. Every conversation bubble around it is identical — it asks the same questions every time. Its memory is a tiny, empty container. The desk is clean — no history, no context. Right side: a sophisticated AI robot with a large, complex glowing brain sits at a desk surrounded by organized files, holographic displays showing customer history, preference maps, and interaction timelines. Its memory container is large and full of organized, color-coded data crystals. It recognizes the person in front of it. The contrast between stateless and stateful is the focal point. Style: clean, modern, side-by-side comparison with distinct color temperatures (cold/empty left, warm/rich right). 16:9 landscape. No text.

## Prompt 3: The Vector Space
An abstract, beautiful illustration of a vast three-dimensional space filled with floating glowing orbs of different sizes and colors. Each orb represents a memory. Orbs that are semantically similar cluster together — a group of blue orbs (billing memories) near each other, a group of green orbs (product memories) nearby but distinct, a group of red orbs (complaint memories) in another cluster. A bright search beam (golden light) enters from one side, and the orbs it passes near light up brighter — showing semantic similarity search in action. The closest orbs glow brightest (most relevant), farther ones dimmer. Lines of light connect related orbs showing relationships. The overall effect is a galaxy of memories being searched by meaning. Style: abstract, cosmic, data visualization aesthetic. Dark space background with glowing elements. 16:9 landscape. No text.

## Usage Notes
- All prompts 16:9 landscape for Medium featured images
- No text, no labels
- Focus on: memory layers, stateless vs stateful, semantic search, knowledge accumulation
- Color coding: blue = working/active, amber = short-term, purple = long-term, gold = retrieval
- Recommended: Prompt 1 (Three Memory Layers) or Prompt 3 (Vector Space) for featured image
