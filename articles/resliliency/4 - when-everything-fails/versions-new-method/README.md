This directory contains versions of the "When Everything Fails" article created using the new human-like writing process.

## Key Differences from Original Process

**Old Process**:
- v1 was a complete draft (1,600 words of full prose)
- Subsequent versions added/refined content
- Started with publishable content

**New Process**:
- v1 is just an outline (structure only)
- v2 adds bullet points and details
- v3 is first full prose (partial)
- v4 is first publishable version
- More natural, iterative progression

## Benefits of New Method

1. **Faster Initial Feedback**: Can review structure (v1) before writing prose
2. **Clearer Progression**: Each version has distinct purpose
3. **Less Wasted Effort**: Don't write full prose for sections that might be cut
4. **More Realistic**: Mirrors how humans actually write
5. **Better for Medium Publisher App**: Clear version-to-version changes for typing simulation
