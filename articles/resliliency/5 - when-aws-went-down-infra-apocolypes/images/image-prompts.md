# Featured Image Prompts for Gemini

## Option 1: Infrastructure Apocalypse (Dramatic)

**Prompt**:
```
Create a dramatic, modern illustration showing a massive data center facility split in half with one side dark and failing (sparks, red error lights, broken servers) and the other side glowing with green success indicators and flowing data streams. In the foreground, show a simplified world map with some regions going dark while others stay illuminated. Use a dark blue/black background with orange/red for failures and cyan/green for success. Style: clean, technical, slightly futuristic, high contrast. No text.
```

**Rationale**: Visually represents the core concept - when infrastructure fails, some systems survive while others don't. The split visual creates immediate impact.

---

## Option 2: Multi-Region Architecture (Technical)

**Prompt**:
```
Create a clean, technical illustration showing three glowing data center nodes connected by network lines forming a triangle. The center node is cracked and dark with red X marks, while the two outer nodes glow bright cyan/blue with green checkmarks and data flowing between them, bypassing the failed center. Above, show a simplified globe with connection lines. Use dark navy background, neon blue/cyan for active systems, red/orange for failures. Style: minimalist, technical diagram aesthetic, isometric perspective. No text.
```

**Rationale**: Directly illustrates multi-region architecture and failover - the technical solution discussed in the article. More educational than dramatic.

---

## Option 3: The AWS Outage Moment (Narrative)

**Prompt**:
```
Create an illustration showing a large AWS logo-inspired cloud shape in the center, fractured and glowing red/orange with lightning bolts and error symbols emanating from it. Around it, show silhouettes of various services - some falling/fading (Netflix, Disney+ icons) and others standing strong with shields (simplified server icons with green force fields). Background: dark gradient from black to deep blue. Style: modern, slightly abstract, dramatic lighting, tech-focused. No text except stylized "us-east-1" in small text on the fractured cloud.
```

**Rationale**: Tells the story of the specific AWS outage event that opens the article. Creates narrative connection and immediate recognition for those who experienced it.

---

## Recommended Choice

**Option 1** - Best balance of visual impact, clarity, and broad appeal. The split visual immediately communicates "some survive, some don't" without requiring technical knowledge to understand.

## Image Specifications

- **Aspect Ratio**: 16:9 (1920x1080) for Medium featured image
- **Style**: Modern, technical, professional
- **Color Palette**: Dark backgrounds with high-contrast neon accents
- **Avoid**: Cluttered details, excessive text, stock photo aesthetics
