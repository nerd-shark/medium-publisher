# Featured Image Prompts — Cross-Functional Communication

## Prompt 1: The Three Languages
Three translucent human silhouettes standing in a triangle formation, each glowing a different color: gold (engineering, with circuit board patterns), blue (product, with chart/metric patterns), and green (design, with user flow/wireframe patterns). Between them, their communication streams (colored light beams) intersect in the center, forming a bright white convergence point — shared understanding. The background is dark. Each figure's "language" (their colored patterns) is distinct but the convergence point shows they can connect. Style: abstract, clean, symbolic. 16:9 landscape. No text.

## Prompt 2: The Parallel Tracks
Three parallel railroad tracks stretching into the distance, each a different color (gold, blue, green) representing engineering, product, and design. The tracks run parallel but never converge — they reach a destination (a product launch platform) separately. The result: three separate, disconnected arrivals. Contrast this with a second set of tracks in the background that merge into one track before reaching the platform — arriving together. Style: dramatic perspective, industrial-modern, dark background. 16:9 landscape. No text.

## Prompt 3: The Rosetta Stone
A futuristic Rosetta Stone tablet floating in space, with three distinct sections glowing different colors. The top section (gold) shows system architecture diagrams and code patterns. The middle section (blue) shows user metrics, charts, and roadmap elements. The bottom section (green) shows user flows, wireframes, and accessibility icons. Where the sections overlap, the symbols transform into a shared language — user stories and business outcomes. Style: ancient-meets-futuristic, dark space background, glowing stone. 16:9 landscape. No text.

## Prompt 4: The Bridge Builders
Three teams on separate floating platforms (gold, blue, green) in a dark void. Each platform has its own tools and artifacts. Between the platforms, bridge segments are being constructed from both sides — meeting in the middle. Some bridges are complete (strong, glowing connections). Others are still being built. The visual metaphor: cross-functional collaboration as bridge-building between disciplines. Style: isometric, clean, architectural. 16:9 landscape. No text.

---

## Usage Notes
- All prompts 16:9 landscape for Medium featured images
- No text, no labels
- Color coding: gold = engineering, blue = product, green = design, white = shared understanding
- Recommended: Prompt 1 (Three Languages) for featured image — clearest visual metaphor for the article's core message
