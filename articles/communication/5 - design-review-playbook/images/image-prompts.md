# Gemini Image Prompts - Design Review Playbook

## Featured Image (Primary)

### Prompt 1: Conference Room Facilitation Scene
```
Create a professional illustration of a design review meeting in a modern conference room. Show a facilitator standing at a whiteboard with a neutral, guiding posture, while 5-6 diverse engineers sit around a table. The facilitator is drawing a simple architecture diagram. Some engineers are engaged and leaning forward, one is taking notes, another is raising their hand to speak. The atmosphere should feel collaborative and productive. Use a clean, modern illustration style with a color palette of blues, grays, and warm accent colors. The whiteboard should show simple boxes and arrows representing a system architecture. Include subtle details like laptops, coffee cups, and notebooks. The lighting should be bright and professional. Style: Modern flat illustration with slight depth, professional but approachable.
```

### Prompt 2: Facilitator vs Advocate Visual Metaphor
```
Create a split-screen illustration showing the difference between a facilitator and an advocate. Left side: A person standing neutrally with open hands, guiding a discussion between two people who are calmly discussing (facilitator). Right side: Two people pointing at each other argumentatively while a third person looks frustrated (advocates without facilitator). Use a clean, modern illustration style with contrasting colors - calm blues and greens on the left (facilitator side), tense reds and oranges on the right (advocate side). Include simple visual elements like speech bubbles, arrows, and abstract shapes to represent ideas. Style: Modern vector illustration, professional but accessible.
```

### Prompt 3: Four Types of Design Reviews Diagram
```
Create an infographic-style illustration showing four connected stages of design reviews. Show four panels arranged in a flow: 1) Requirements DR with people around a table with question marks, 2) Preliminary DR with someone drawing high-level architecture on a whiteboard, 3) Detailed DR with engineers looking at detailed diagrams and code, 4) Final DR with a checklist and deployment symbols. Use a progression of colors from light blue (requirements) to dark blue (final). Include simple icons for each stage: lightbulb for requirements, blueprint for preliminary, magnifying glass for detailed, checkmark for final. Style: Clean infographic design, modern and professional, suitable for Medium article header.
```

### Prompt 4: Bikeshedding Visual Metaphor
```
Create a humorous but professional illustration showing the concept of bikeshedding in design reviews. Show a large, complex database architecture diagram in the background (faded, ignored), while in the foreground, multiple people are intensely debating over a tiny detail like variable naming (shown as "userId vs user_id" on a small sticky note). Use visual scale to emphasize the contrast - the important decision is huge but faded, the trivial decision is small but bright and in focus. Include frustrated body language and exaggerated expressions. Use a color palette that emphasizes the contrast - muted grays for the important background, bright colors for the trivial foreground debate. Style: Modern illustration with a touch of humor, professional but relatable.
```

### Prompt 5: Decision Framework Comparison
```
Create a clean comparison diagram showing three decision-making frameworks: Consensus (everyone agrees - show multiple people all nodding with checkmarks), Consent (no blocking objections - show people with thumbs up and one person neutral but not blocking), and DACI (clear roles - show four distinct roles with labels: Driver, Approver, Contributors, Informed). Use a three-column layout with simple icons and minimal text. Color code each framework: green for consensus, blue for consent, purple for DACI. Include simple human figures in different poses to represent agreement, neutrality, and role clarity. Style: Clean infographic design, modern and minimalist, suitable for social media sharing.
```

## Alternative Concepts

### Concept A: Journey/Process Visualization
```
Create a visual journey showing the progression through design reviews. Show a winding path with four stops (Requirements DR, Preliminary DR, Detailed DR, Final DR), with small illustrations at each stop showing what happens at that stage. Use a map-like aesthetic with a clear start and end point. Include visual indicators of risk level (high-risk path uses all stops, low-risk path shortcuts). Style: Modern journey map illustration, clean and professional.
```

### Concept B: Before/After Comparison
```
Create a before/after comparison showing a design review without facilitation (chaotic, unproductive) versus with facilitation (organized, productive). Left side: Messy meeting with people talking over each other, no decisions, frustrated expressions. Right side: Organized meeting with clear structure, engaged participants, visible decisions being documented. Use contrasting colors and visual chaos vs order. Style: Split-screen illustration, modern and clear.
```

### Concept C: Facilitator Toolkit
```
Create an illustration showing a "facilitator's toolkit" with visual representations of key techniques: time-boxing (clock), parking lot (notepad), decision frameworks (flowchart), psychological safety (shield), documentation (checklist). Arrange these tools in an organized, visually appealing layout. Use icons and minimal text. Style: Modern icon set illustration, clean and professional.
```

## Image Specifications

### Dimensions by Platform
- **Medium Featured Image:** 1400x788 (16:9 ratio)
- **LinkedIn Article:** 1200x627 (1.91:1 ratio)
- **Twitter/X Card:** 1200x675 (16:9 ratio)
- **Facebook Post:** 1200x630 (1.91:1 ratio)
- **Instagram Post:** 1080x1080 (1:1 ratio)
- **Instagram Story:** 1080x1920 (9:16 ratio)

### Style Guidelines
- **Color Palette:** Blues (#2C5F8D, #4A90E2), Grays (#F5F5F5, #333333), Accent colors (warm orange #FF6B35 for emphasis)
- **Typography:** Clean, modern sans-serif fonts
- **Tone:** Professional but approachable, not corporate or stuffy
- **Avoid:** Stock photo look, overly corporate aesthetics, clipart style

### Accessibility Considerations
- High contrast between text and background
- Readable at small sizes
- Color-blind friendly palette
- Clear visual hierarchy

## Recommended Primary Image

**For Medium Article:** Use Prompt 1 (Conference Room Facilitation Scene) or Prompt 3 (Four Types of Design Reviews Diagram)

**Rationale:** 
- Prompt 1 creates an immediate visual connection to the article's content (facilitation in action)
- Prompt 3 provides educational value and is highly shareable
- Both are professional, clear, and work well across platforms

**For Social Media:** Use Prompt 4 (Bikeshedding Visual Metaphor) for engagement-focused posts, or Prompt 5 (Decision Framework Comparison) for educational posts
