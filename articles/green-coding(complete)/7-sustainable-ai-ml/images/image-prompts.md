## Image Selection Criteria

When choosing between the generated images, consider:

1. **Immediate Recognition**: Does the image clearly communicate "AI/ML + Sustainability"?
2. **Professional Quality**: Is it suitable for LinkedIn, Medium, and professional contexts?
3. **Visual Impact**: Does it stand out in a social media feed?
4. **Clarity at Small Sizes**: Does it remain clear when viewed as a thumbnail?
5. **Brand Consistency**: Does it match the style of other articles in the series?
6. **Emotional Resonance**: Does it evoke the right feeling (optimistic, technical, actionable)?

## Usage Guidelines

**File Naming Convention:**
- `featured-image.png` - Primary featured image
- `featured-image-square.png` - Square crop for Instagram (1080x1080)
- `featured-image-vertical.png` - Vertical crop for Stories (1080x1920)

**Optimization:**
- Compress images to <500KB for web performance
- Maintain high quality for featured image placement
- Create multiple sizes for different platforms

**Attribution:**
- If using Gemini-generated images, note: "Image generated with Google Gemini"
- Include in article metadata if required by platform
