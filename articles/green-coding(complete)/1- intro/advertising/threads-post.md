That API call you just made has the same carbon footprint as boiling water for tea ☕

Software runs on physical servers. Global data centers = 1-2% of worldwide electricity. If the internet were a country, it'd rank 6th in energy consumption.

Green coding = writing efficient software that minimizes energy consumption.

Why care?
💰 Energy efficiency = lower AWS bills
⚡ Efficient code is better code (faster, more scalable)
🌍 Your code runs millions of times—small changes = massive impact

Real example: Switching List to HashSet for lookups
• 10-minute fix
• 100x performance improvement  
• 40% CPU reduction
• Same result, fraction of the cost

Quick wins you can do today:
✅ Delete unused cloud resources
✅ Optimize images (5MB → 500KB)
✅ Right-size infrastructure
✅ Check your region's carbon intensity
✅ Improve caching strategy

Fun fact: C is 75x more energy efficient than Python. But don't rewrite everything—just be intentional about where efficiency matters.
